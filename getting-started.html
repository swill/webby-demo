Setup Guide

Getting Started
with Webby

From zero to a live, editable website in under 30 minutes. No command line. No deployment scripts. Just a browser, a GitHub account, and an Anthropic key.

Step One

Design Your
Initial Prompt

Webby sites begin life as HTML generated by Claude. The Bootstrap Prompt is a carefully crafted template that tells Claude exactly what kind of site to build for you.

Open the Bootstrap Prompt from the Webby GitHub repository. Fill in the bracketed fields — your business name, profession, tagline, location, and contact details. Attach 2–5 inspiration images that capture the visual mood you're after.

Paste the completed prompt into Claude.ai and Claude will generate a complete, multi-page, Webby-ready HTML site. All data-zone, data-editable, and data-editable-image attributes will be in place — ready to load straight into edit mode.

View the Bootstrap Prompt →

Tips for a Great Prompt

  • 🎨   Attach 2–5 images that represent your visual style
  • 📝   Be specific about your profession and audience
  • 🏷️   Write a memorable tagline — it becomes your hero headline
  • 📄   List all the pages you need (home, about, services…)
  • 🔗   Include your real contact details and social links

The prompt is designed to produce output with these attributes already in every section:

data-zone data-zone-label data-editable data-editable-image

Step Two

Sign Up to
Free Services

Webby is built on free tiers of two services you may already use. Both are required to enable publishing and AI features.

🐙

GitHub (Free)

GitHub hosts your HTML files and deploys them automatically to GitHub Pages. You need a free account, a public repository, and a fine-grained Personal Access Token (PAT) scoped to contents: read + write for that repository.

  • ✦   Create a free account at github.com
  • ✦   Create a new public repository
  • ✦   Enable GitHub Pages (Settings → Pages → main / root)
  • ✦   Generate a fine-grained PAT with contents: read + write
🤖

Anthropic (Pay-as-you-go)

The AI section generator uses Claude via the Anthropic API. A small amount of free credit is included when you sign up; generating a typical section costs only a fraction of a cent, making this effectively free for most users.

  • ✦   Sign up at console.anthropic.com
  • ✦   Navigate to API Keys and create a new key
  • ✦   Copy your sk-ant-... key for use in secrets.js
  • ✦   Usage is billed only for what you generate

Step Three

Configure
secrets.js

Create a file called secrets.js in the same folder as your index.html. This file never gets published — it stays on your computer and activates edit mode locally.

The file tells Webby your Anthropic API key (for AI generation), your GitHub token (for publishing), and which repository and branch to push to.

⚠️ Security Note

secrets.js is never included in published output. Your local folder is not a git repo — Webby uses the GitHub API directly. Your keys go nowhere except to the GitHub and Anthropic APIs.

// secrets.js — stays local only
window.SITE_SECRETS = {
  anthropicKey: "sk-ant-...",
  githubToken: "ghp_...",
  repo: "username/repo-name",
  branch: "main"
};

Your Local Folder Structure

📁 my-site/
   📄 index.html
   📄 secrets.js (local only)
   📁 assets/
      🖼️ placeholder.jpg

Step Four

Open & Start Editing

With your HTML and secrets.js in the same folder, open index.html in your browser. The editor toolbar activates automatically. Click any text to edit it, click any image to replace it — it's that immediate.

✏️

Click Text to Edit

Any element with data-editable becomes editable on click. Type directly on the page — changes are live instantly.

🖼️

Click Images to Replace

Images with data-editable-image open a file picker on click. The chosen file uploads directly to your GitHub repo and the page updates immediately.

✨

AI-Generate New Sections

Hover between sections and click the + Add Section button. Describe what you want — Webby's AI writes and inserts a fully styled HTML section.
