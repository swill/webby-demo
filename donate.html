Philosophy

Free & Open Source.
Enabling Creativity.

Webby exists because I believe that publishing a beautiful website should be accessible to everyone — not just developers, and not just those who can afford a monthly CMS subscription.

Every line of code is open source and available on GitHub. There is no paid tier, no feature gating, and no plan to add one. The full functionality — inline editing, AI generation, publishing — is free for personal and commercial use under the MIT License.

The project is built on my own time, fuelled by genuine curiosity about what the web can be when the friction of publishing is removed. Your support, whether financial or a GitHub star, means more than you might expect.

Star on GitHub ⭐

MIT Licensed

Use Webby for any project — personal, client, commercial. Modify it, fork it, build on it. The only ask is attribution.

No Vendor Lock-in

Your site is just an HTML file on GitHub. If you ever want to leave, everything you've built is portable. No exports, no migrations, no data loss.

Community First

Issues, feature requests, and pull requests are welcome. Webby improves every time someone uses it and shares what they found.
